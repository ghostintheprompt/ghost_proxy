## Installation

### macOS (Recommended)
Download the latest DMG from the releases page: https://github.com/ghostintheprompt/ghost-proxy/releases

```bash
# Or via Homebrew
brew install --cask ghost-proxy
```

### Build from Source
```bash
git clone https://github.com/ghostintheprompt/ghost-proxy.git
cd ghost-proxy
npm install
npm run dev
```

## Privacy Statement
Ghost Proxy is designed with a Zero-Telemetry architecture.
- **Local Only:** All scripts, logs, and configurations are stored locally in your browser/machine.
- **No Cloud:** Aside from the optional Gemini API link for script generation, no data ever leaves your device.
- **No Tracking:** We do not include analytics, tracking pixels, or third-party cookies.
